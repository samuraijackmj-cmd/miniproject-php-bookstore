# 📚 Mini Project: Online Bookstore

โปรเจคนี้เป็นระบบร้านขายหนังสือออนไลน์  
พัฒนาโดยใช้ PHP และ MySQL  
เป็นโปรเจคสำหรับฝึกทำ Web Application แบบครบวงจร

---

## 🔧 ฟีเจอร์หลัก

- สมัครสมาชิก / เข้าสู่ระบบ
- แสดงรายการหนังสือ
- เพิ่มสินค้าใส่ตะกร้า
- ระบบสั่งซื้อสินค้า
- ประวัติการสั่งซื้อ
- ระบบ Wishlist
- ระบบหลังบ้าน (Admin Panel)
  - จัดการหนังสือ
  - จัดการหมวดหมู่
  - จัดการคำสั่งซื้อ
  - รายงานยอดขาย

---

## 🛠 เทคโนโลยีที่ใช้

- PHP (Core PHP)
- MySQL
- HTML / CSS
- JavaScript
- AJAX
- XAMPP

---

## ⚙ วิธีติดตั้ง

1. Clone โปรเจค

git clone https://github.com/samuraijackmj-cmd/miniproject-php-bookstore.git


2. ย้ายโฟลเดอร์ไปไว้ใน

C:\xampp\htdocs\


3. สร้างฐานข้อมูลใน phpMyAdmin  
4. Import ไฟล์ SQL  
5. แก้ไฟล์เชื่อมต่อฐานข้อมูลในโฟลเดอร์ `config`  

เข้าใช้งานที่:

http://localhost/bookstore
